# Prompt Engineering and Agent Origins

本方向收录 Prompt Engineering、早期 Agent 方法和工具使用相关论文，例如 Chain-of-Thought、Self-Consistency、ReAct、Tool Use、Planning、Reflection、Tree Search 等。该方向重点关注大语言模型如何通过提示设计、推理轨迹、工具交互、规划搜索和自我反思，逐步发展为具备任务执行能力的语言智能体。
