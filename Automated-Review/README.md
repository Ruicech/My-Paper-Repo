# Automated Review

本方向是课题组当前主线，覆盖自动化评审及其上下游任务，包括审稿生成、评审质量评估、Meta-review、Rebuttal、Reviewer assignment、LLM-as-reviewer、评审一致性与可靠性分析。

随着当前课题逐渐收敛到“论文创新性 / 新颖性评价”，本目录下新增 `novelty-assessment` 子方向，用于单独整理创新性分析、novelty critique、相关工作对比、贡献分解、证据支撑检查以及 LLM novelty evaluation benchmark 等论文。

主目录的论文清单只保留通用自动化评审、科研自动化、审稿生成与评审质量评估相关论文；已经归入 `novelty-assessment` 子方向的论文不在主表中重复收录。
