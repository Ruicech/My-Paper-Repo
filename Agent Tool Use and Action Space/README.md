# Agent Tool Use and Action Space

本方向收录 LLM Agent 工具使用与动作空间相关论文，重点关注 Tool Use、Function Calling、API Calling、Code-as-Action、代码执行环境、工具链组合、执行反馈、自我修正与复杂任务执行等问题。

该方向主要回答的问题是：Agent 如何与外部工具、API、代码解释器、浏览器、数据库或其他环境进行交互，并通过这些交互完成复杂任务。
