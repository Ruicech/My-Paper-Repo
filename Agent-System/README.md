# Agent System

本方向收录 Agent 系统架构与工程实现相关论文，重点关注 Agent 执行框架、工具调用、任务规划与分解、多 Agent 协作、模型路由、上下文管理、轻量化部署、本地运行和推理效率优化等问题。

该方向和自动化评审的关系主要包括：构建可执行的论文评审 Agent 系统，将论文解析、证据检索、方法分析、实验检查、问题生成、评分与总结等环节组织为可控工作流；通过模型路由和任务分解，让简单任务由低成本模型完成，复杂或高风险任务由更强模型、多 Agent 协作流程或人工复核完成；同时支持敏感论文数据的本地化处理与评审过程追踪。
